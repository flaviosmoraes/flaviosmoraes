## 🚀 Projetos em Destaque

### 🔥 Plataforma IoT de Monitoramento de Gás
> Projeto acadêmico de ponta a ponta — do hardware ao software em produção na nuvem

- Dispositivo físico com **NodeMCU (ESP8266)** + sensor **MQ-2**, programado em **C++**
- Transmissão de dados em tempo real via Wi-Fi para endpoint na nuvem
- Back-end em **Python/Flask** + banco de dados **MongoDB**
- Dashboard web com visualização de histórico, análise de tendências e **alertas automáticos**
- Infraestrutura em **Oracle Cloud (OCI)** com duas VMs dedicadas — uma para o endpoint IoT, outra para a aplicação web

**Stack:** `C++` `Python` `Flask` `MongoDB` `JavaScript` `Oracle Cloud` `NodeMCU`
